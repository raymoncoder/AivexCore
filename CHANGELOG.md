## [0.1.0] — 2025-03-06

### 🚀 Initial Release

#### Core Components (26)
- `BootLoader` — Cinematic boot sequence animation
- `CustomScrollBar` — Styled scrollbar with emerald accent
- `DragDropCard` — Drag & drop list interface
- `FileUploadZone` — Premium file upload with drag-and-drop
- `NeuralAccordion` — Animated accordion with smooth transitions
- `NeuralAlert` — Contextual feedback with success/error/warning variants
- `NeuralAvatar` — User avatar with status indicator
- `NeuralBadge` — Status badge with animated dot
- `NeuralBreadcrumb` — Navigation breadcrumb
- `NeuralButton` — Primary button with neon/ghost/secondary variants
- `NeuralCard` — Composable card primitive
- `NeuralCheckbox` — Accessible checkbox with animation
- `NeuralDatePicker` — Calendar date picker
- `NeuralDialog` — Accessible modal with backdrop blur
- `NeuralInput` — Premium text input with label and error states
- `NeuralLoader` / `NeuralLoaderVariants` — 6 loading animation styles
- `NeuralMarquee` — Infinite scroll marquee
- `NeuralNumberInput` — Number input with increment/decrement
- `NeuralOTPInput` — One-time password input
- `NeuralPasswordInput` — Password field with toggle visibility
- `NeuralProgress` — Gradient progress bar with glow
- `NeuralSearchInput` — Search field with icon
- `NeuralSelect` — Custom dropdown select
- `NeuralSkeleton` — Loading skeleton
- `NeuralSlider` — Range slider
- `NeuralSwitch` — Toggle switch (Radix)
- `NeuralTabs` — Tabs with default/pill/underline variants
- `NeuralTextarea` — Multiline text input
- `NeuralToaster` — Toast notification system

#### AI Components (6)
- `AIChatInterface` — Full chat UI with streaming support
- `AgentActivityFeed` — Real-time agent activity log
- `AgentThoughtFlow` — Agent reasoning visualization
- `ModelParameters` — AI model configuration panel
- `StepProcess` — Multi-step process indicator
- `StreamingText` — Typewriter streaming text effect

#### Crypto Components (5)
- `AddressBadge` — Blockchain address display with copy
- `PriceMetric` — Token price with change indicator
- `SwapCard` — DeFi token swap interface
- `TokenPerformance` — Token performance chart
- `WalletConnectModal` — Multi-wallet connection modal

#### Data & Visualization (7)
- `AnalyticsCard` — Metric card with sparkline
- `CommandPalette` — ⌘K command palette
- `MiniSparkline` — Inline sparkline chart
- `NeuralCodeEditor` — Syntax-highlighted code editor
- `NeuralDataTable` — Sortable, paginated data table
- `NeuralExportButton` — Data export (CSV/JSON)
- `RadarChart` — Radar/spider chart

#### Effects & Patterns (5)
- `Magnetic` — Magnetic cursor effect
- `NeuralBentoGrid` / `NeuralBentoCard` — Bento layout system
- `NeuralCarousel` — Component carousel
- `NeuralCommandBar` — Floating command bar
- `SpotlightCard` — Card with cursor spotlight effect
